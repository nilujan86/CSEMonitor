# CSE Monitor — Android App & Home Screen Widget

Real-time Colombo Stock Exchange stock tracker with a live home screen widget.

---

## Features

| Feature | Detail |
|---|---|
| **Watchlist** | Add/remove any CSE-listed stock by symbol |
| **Live prices** | Fetches from `cse.lk` API — same source as the official portal |
| **Intraday chart** | MPAndroidChart line graph with touch zoom/pan |
| **ASPI banner** | All Share Price Index displayed in the toolbar |
| **Home-screen widget** | Resizable 3×2 widget — shows price + % change, tap to open detail |
| **Auto-refresh** | Every 60 s during market hours (Mon–Fri 09:30–14:30 IST), 5 min otherwise |
| **Dark theme** | Full dark UI, green/red gain/loss colouring |
| **Offline cache** | Room DB keeps last-known prices between sessions |

## Quick Start

### Prerequisites
- Android Studio Hedgehog (2023.1.1) or newer
- JDK 17
- Android SDK 34 (API 34)
- Minimum device: Android 8.0 (API 26)

### Build

```bash
# 1. Clone / extract project
cd CSEMonitor

# 2. Open in Android Studio → File → Open → select this folder
#    OR build from command line:

# On macOS/Linux
./gradlew assembleDebug

# On Windows
gradlew.bat assembleDebug

# 3. Install on device / emulator
./gradlew installDebug
```

> **First build** downloads ~150 MB of dependencies (Gradle, AndroidX, Retrofit, MPAndroidChart).
> Subsequent builds are cached and much faster.

### Generate launcher icons (one-time)
```bash
pip install Pillow
python3 generate_icons.py
```

## How the Widget Works

1. Long-press home screen → **Widgets** → drag **CSE Stock Widget** onto the screen.
2. A config screen appears — pick which stock to display.
3. The widget refreshes automatically. You can also tap the **⟳** icon in the top-right corner.
4. To change the pinned stock: open the app → tap a stock → **⋮ menu → Pin to widget**.

## API Notes

The app uses the **unofficial CSE web API** at `https://www.cse.lk/api/`.
This is the same JSON API consumed by the official CSE website.

Key endpoints used:

| Endpoint | Method | Purpose |
|---|---|---|
| `companyInfoSummery` | POST (form) | Stock price + company info |
| `marketSummary` | GET | ASPI, S&P SL20, turnover |
| `intradayPriceData` | POST (form) | Tick-by-tick intraday series |
| `symbolSearch` | POST (form) | Company name / symbol search |

> **Note:** The CSE API is not officially documented. If the app returns errors, the API
> endpoint paths or field names may have changed. Check the CSE website's network tab
> in browser DevTools to verify current endpoints.

## Customisation Tips

- **Add more stocks:** edit `CSE_POPULAR_STOCKS` in `StockModels.kt`.
- **Change refresh interval:** edit `delay()` values in `StockUpdateService.kt`.
- **Widget size:** adjust `minWidth`/`minHeight` in `res/xml/stock_widget_info.xml`.
- **Market hours:** edit `isMarketHours()` in `StockUpdateService.kt` if CSE changes trading hours.
